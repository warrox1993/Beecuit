# Au Fil des Saveurs

Boutique e-commerce belge premium pour biscuits artisanaux — full-stack Next.js 15.

> Working repo name `beecuit` and Vercel deployment `beecuit.vercel.app` are kept
> until the commercial domain is acquired. See `docs/style-guide.md` for the brand system.

## Stack
Next.js 15 · TypeScript · Tailwind v4 · shadcn/ui · Drizzle ORM · Neon · Auth.js v5 · next-intl · Stripe · Vercel
